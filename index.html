What is a Function

A function is a fundamental building block in programming, which takes input, performs a specific operation on it, and returns an output.
Functions enable the reuse of code and promote modularity, making it easier to manage and understand complex programs.
Or A function is a block of code that performs a specific task. It's a reusable piece of code that can take inputs (arguments), execute a set of statements, and return a value.

Function Call

A function call is when you invoke a function to execute its code. In this example, we call the `greet` function with the argument `"Kuhle"` and log the returned value to the console.
For example - let result = greet("Kuhle"); console.log(result); // Output: Hello, Kuhle

Function Definition

A function definition is the actual code that makes up the function. It includes the function name, parameters, and the code that gets executed when the function is called
For Example: function greet(name) {
console.log(`Hello, ${name}!`);
}

Variables

Variables are containers for storing data values. In JavaScript, variables are declared using var, let, or const keywords, followed by the variable name
Example : let myName = 'Kuhle';
let x = 10;
const PI = 3.14;
var message = "Hello, World!";

Data Types

Data types determine the type of value a variable can hold. JavaScript has several data types, including:
- Number (e.g., 42)
- String (e.g., 'hello')
- Boolean (e.g., true or false)
- Null (e.g., null)
- Undefined (e.g., undefined)
- Object (e.g., { name: 'Kuhle', age: 27 })
- Array (e.g., [1, 2, 3])
